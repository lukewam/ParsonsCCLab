ASSIGNMENT 05

OPTICAL ILLUSION

A05

Project Reflection

My research focused on how to make optical illusions feel alive through motion. A lot of classic illusions are static images that appear to move, but I was interested in illusions that are actually animated—patterns that create a strong mental or physical effect when they flicker on the screen.

At first, I tried to build the piece using a custom class structure, similar to one of the examples we saw. When I started writing it out, I realized it was more complicated than it needed to be. For this kind of work, we often care less about optimization or Big-O, and more about clarity: if the visual effect is strong, the code doesn’t need to be overly engineered.

The final sketch is simple: a grid of small rectangles, each assigned a random color every frame. The vertical bars slide horizontally, while the colors keep flashing. Together, they create a strong, disturbing effect that can make people feel slightly dizzy or unsettled.

I can imagine this pattern being used in a video or installation as a background texture to amplify tension—something that quietly pushes the mood into an uneasy, unstable space.
