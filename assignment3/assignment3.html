ASSIGNMENT 03

EXPERIMENTAL CLOCK

A03

Project Reflection

This clock is inspired by a kind of quiet, romantic oriental elegance — the idea that time should not only exist as numbers, but as part of nature and art. In nature, flowers and leaves are one way we feel time passing: they are seasonal, and the sun shifts across the sky to mark the day. I wanted the clock to feel like a single painting: no numbers, no clock hands, just an image that carries time inside it.

In this project, the leaves represent the hours, the flowers represent the minutes, and the sun represents day and night. The number of leaves equals the current hour in a 12-hour format. If the sun is visible, it is the first 12 hours of the day; if there is no sun, it is the second 12 hours. The number of flowers shows how many 5-minute segments have passed within the current hour, like the markings on a traditional clock.

For example, in the morning, 8:08 would be represented as: a sun (it is in the first half of the day), eight leaves, and two flowers.

I approached this assignment through several ideas, starting from more conventional representations — using numbers and clock hands — and then gradually moving away from them. I kept asking how I could represent time differently, in a way that feels more like an atmosphere than an interface.

What I learned from this assignment is that code is just one tool in the larger process of making an artwork. The code itself is not very complex, but it enables a visual system that feels poetic and complete on the screen.
